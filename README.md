### 1. 성향(MBTI) 입력
화면 상단의 직관적인 그리드 인터페이스를 통해 본인의 MBTI를 직접 선택합니다.

### 2. 감성 질문 답변 (3문항)
너무 무겁지 않으면서도 당신의 속마음과 인생관을 툭 끄집어내는 3가지 질문에 답합니다.

심정 (BGM): 지금 내 마음에 흐르는 감정의 주파수 선택

취향 (무드): 오늘 밤 인스타 피드에 남기고 싶은 시각적 감성 선택

가치관 (사과): 내가 인생에서 가장 베어 물고 싶은 가치(사랑/성공/영감/치유) 선택

### 3. 나만의 '시적 페르소나'와 인생시 배달
분석이 끝나면 '고독한 나타샤', '사색하는 별빛' 등 당신만의 시적 페르소나가 부여됩니다.

입력한 MBTI 점수와 3가지 감성 키워드 가산점을 합산하여, 지금 이 순간 당신에게 가장 필요한 '인생시'가 매칭 사유와 함께 배달됩니다.

## 알고리즘 가이드 (매칭 엔진)
MBTI 매칭 점수: 입력한 유형과의 기본 적합도 반영 (0 ~ 4점)

감성 질문 가산점: 3가지 감성 질문 선택지에 매핑된 정서적 키워드가 일치할 때마다 문항당 +1.5~2.0점 부여

최종 스코어: 합산 점수가 가장 높은 시를 우선순위로 추천 리스트에 노출

## 결과 공유하기
결과 페이지 하단의 공유하기 버튼을 통해 당신의 시적 페르소나와 인생시 구절을 친구들에게 공유하고, 오늘 밤의 감성을 함께 나누어 보세요!
